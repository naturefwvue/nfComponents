# 自然框架之前端组件 
natureFramework Components  
vue + html5 做的一套表单、查询、列表组件，练手用。  
熟悉vue的使用，顺便做一个元数据的生成、维护工具

# 环境
vue3.0 

# 安装

npm i 

# 运行

## Project setup
```
npm install
```

npm run serve
